# ciclope_mitologias_verbales
Si buscas herramientas para sostener la tensión entre marcos incompatibles, bienvenido al Cíclope. Escribimos con máquinas, con archivos, con modelos, con tradiciones que ya no se distinguen claramente de los sistemas que las procesan. La pregunta ya no es quién escribió esto, sino qué régimen de lectura estamos activando cuando leemos.
